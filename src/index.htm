0. The 3D printers that Node Pro can be adapted to

  • Ender3 v1/v2
  • Ender3 Pro (use GD32,need special SD plug)
  • Ender5,Ender5 S1, Ender7
  • Neptune2, Neptune 3 Pro
  • Kingroon KP3S(not support KP3S Pro)
  • Anycubic Kobra Go
  • Prusa Mk3S/MK3S+

If your printer is not in the above list, we do not recommend using Node Pro at this time.
